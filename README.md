## AR triangle generator

삼각형모양을 AR을 통해 영상에 있는 모니터로 켠 체스판에 출력해주는 간단한 프로젝트

## 사용법

삼각형모양을 넣어주고싶은 영상을 찍어 chessboard.avi를 덮어씌워준다.
이때 8x5체스보드가 찍혀있어야한다.
